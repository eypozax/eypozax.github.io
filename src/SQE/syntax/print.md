To print, ```insert``` method is used:

```
insert <TEXT>
```

```
insert Hello World!
```

<img src="/assets/images/single-line-insert.webp" alt="Before">

To print multiple line text, use curly brackets:
```
insert {
<MULTIPLE LINES>
}
```

```
insert {
line 1
line 2 
}
```
